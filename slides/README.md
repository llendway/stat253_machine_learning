## Instructions

You can open these slides and see presenter notes by pressing "p". To see the presenter notes on another screen, add ?presentme=true to the URL of the presentation, for example 

my-presentation.html?presentme=true

See more details in Yihui Xie, J. J. Allaire, Garrett Grolemund's book [here](https://bookdown.org/yihui/rmarkdown/ioslides-presentation.html#presenter-mode).

You can also follow along with the YouTube videos [here](https://youtube.com/playlist?list=PLyEH7o09I466RkmttSwRoEU8JufJXS2m3).
